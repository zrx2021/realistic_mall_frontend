## 🚀 部署说明

### 前端部署
1. 代码已集成到主项目
2. 路由已配置：`/goods/brand`
3. 组件已注册
4. API已定义

### 后端对接要求

#### 1. 品牌列表接口
```
POST /api/brands/list?pageNumber=1&pageSize=10
Body: { "keyword": "搜索词", "country": "国家", "isActive": true }
Response: PageResponse<BrandInfo>
```

#### 2. 品牌CRUD接口
```
GET    /api/brands/{id}          # 获取详情
POST   /api/brands               # 新增品牌
PUT    /api/brands/{id}          # 更新品牌
DELETE /api/brands/{id}          # 删除品牌
```

#### 3. 图片上传接口
```
POST /api/file/upload/image
FormData: { file: File, category: "goods brand" }
Response: "/file/image/goods brand/2025/01/16/xxx.png"
```

#### 4. 其他接口
```
POST /api/brands/batch/delete    # 批量删除
PUT  /api/brands/{id}/status      # 更新状态
GET  /api/brands/export           # 导出数据
GET  /api/brands/active           # 获取启用品牌
```

### 环境要求
- Node.js 16+
- Vue 3.x
- TypeScript 5.x
- Vite 6.x
