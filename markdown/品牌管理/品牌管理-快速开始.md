# 品牌管理 - 快速开始指南

## 🚀 快速访问

访问路径：`/goods/brand`

页面名称：品牌管理 (BrandManagement)

## 📋 功能清单

### ✅ 已实现功能

1. **品牌列表展示**
   - 表格形式展示所有品牌
   - 支持分页（10/20/50/100条/页）
   - 支持排序（按排序权重、创建时间）

2. **品牌搜索**
   - 搜索品牌名称
   - 搜索英文名称
   - 实时搜索反馈

3. **品牌添加**
   - 点击"新建品牌"按钮
   - 填写品牌信息
   - 上传品牌LOGO
   - 保存品牌数据

4. **品牌编辑**
   - 点击表格中的"编辑"按钮
   - 修改品牌信息
   - 更新品牌数据

5. **品牌删除**
   - 点击表格中的"删除"按钮
   - 二次确认
   - 删除品牌数据

6. **状态管理**
   - 启用/禁用品牌
   - 即时生效
   - 状态同步

7. **数据导出**
   - 导出当前搜索结果
   - Excel格式

## 🎯 使用场景

### 场景1: 添加新品牌

**步骤**：
1. 访问 `/goods/brand`
2. 点击左上角"新建品牌"按钮
3. 填写表单：
   - 品牌名称（必填）：例如 "苹果"
   - 品牌英文名（可选）：例如 "Apple"
   - 上传LOGO（可选）：点击上传区域选择图片
   - 品牌描述（可选）：例如 "美国知名科技公司"
   - 品牌官网（可选）：例如 "https://www.apple.com"
   - 品牌国家（可选）：从下拉列表选择 "美国"
   - 排序权重（必填）：例如 100（数值越大越靠前）
   - 启用状态（默认启用）
4. 点击"确定"保存
5. 查看成功提示，列表自动刷新

### 场景2: 编辑现有品牌

**步骤**：
1. 在品牌列表中找到要编辑的品牌
2. 点击该行的"编辑"按钮
3. 弹窗自动填充该品牌的所有信息
4. 修改需要更新的字段
5. 点击"确定"保存
6. 查看成功提示，列表自动刷新

### 场景3: 搜索品牌

**步骤**：
1. 在右上角搜索框输入关键词
2. 点击搜索图标或按回车键
3. 查看搜索结果
4. 点击清空按钮可以重置搜索

### 场景4: 禁用品牌

**步骤**：
1. 找到要禁用的品牌
2. 点击"状态"列的开关按钮
3. 开关变为"禁用"状态
4. 查看成功提示
5. 禁用的品牌不会在前台展示

### 场景5: 删除品牌

**步骤**：
1. 找到要删除的品牌
2. 点击该行的"删除"按钮
3. 确认删除提示弹窗
4. 点击"确定"
5. 品牌被删除，列表自动刷新

⚠️ **注意**：删除操作不可恢复，请谨慎操作

## 📊 数据字段说明

### 必填字段
- **品牌名称**：1-100个字符，唯一
- **排序权重**：0-9999的数字

### 可选字段
- **品牌英文名**：最多200个字符
- **品牌LOGO**：图片文件，≤2MB
- **品牌描述**：最多500个字符
- **品牌官网**：URL格式
- **品牌国家**：从预设列表选择
- **启用状态**：默认启用

## 🖼️ 图片上传说明

### 上传要求
- **格式**：JPG、PNG、GIF、WEBP
- **大小**：不超过 2MB
- **建议尺寸**：200x200 像素
- **比例**：建议 1:1（正方形）

### 上传步骤
1. 点击上传区域
2. 选择本地图片文件
3. 等待上传完成
4. 预览已上传的图片
5. 如需更换，悬停后点击"移除"按钮

### 图片预览
- 列表页显示尺寸：60x60px
- 弹窗预览尺寸：104x104px
- 支持悬停放大查看

## 🔍 搜索说明

## 🐛 常见问题

### Q1: 品牌名称重复怎么办？
**A**: 品牌名称在数据库中是唯一的，如果尝试添加重复的品牌名称，系统会提示错误。建议使用不同的名称或在名称中加入区分标识。

### Q2: 上传的图片不显示？
**A**: 请检查：
1. 图片格式是否正确（JPG/PNG/GIF/WEBP）
2. 图片大小是否超过2MB
3. 网络连接是否正常
4. 后端上传接口是否正常

### Q3: 删除品牌后能恢复吗？
**A**: 删除操作是永久性的，删除后无法恢复。建议在删除前：
1. 确认该品牌确实不再使用
2. 先禁用品牌而不是删除
3. 导出数据做备份

### Q4: 如何批量管理品牌？
**A**: 当前版本不支持批量操作，未来版本将支持：
- 批量删除
- 批量启用/禁用
- 批量导入

### Q5: 品牌的排序权重如何设置？
**A**: 
- 数值范围：0-9999
- 数值越大，品牌在列表中越靠前
- 建议按重要程度设置：
  - 核心品牌：100-999
  - 重要品牌：50-99
  - 普通品牌：10-49
  - 其他品牌：0-9

## 💡 使用技巧

### 技巧1: 快速创建品牌
只需填写必填字段（品牌名称、排序权重）即可快速创建品牌，其他信息可以后续补充。

### 技巧2: 使用排序权重管理展示顺序
合理设置排序权重，可以控制品牌在列表中的展示顺序，无需手动调整。

### 技巧3: 先禁用再删除
对于暂时不需要的品牌，建议先禁用而不是删除，以便将来恢复使用。

### 技巧4: 善用搜索功能
当品牌数量较多时，使用搜索功能可以快速定位目标品牌。

### 技巧5: 定期导出备份
定期导出品牌数据，做好数据备份，防止误删除。
