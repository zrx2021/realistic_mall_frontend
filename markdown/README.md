## 🔖 文档编写规范

### 文档命名规范
- 使用有意义的文件名
- 使用中划线分隔多个单词
- PRD 文档以 `PRD-` 开头
- README 文档以 `README` 开头

### 文档结构规范
- 使用 Markdown 格式
- 清晰的标题层级
- 必要的目录索引
- 代码示例使用代码块
- 表格展示结构化数据

### 文档更新规范
- 更新文档时注明更新日期
- 重大变更需要在文档顶部说明
- 保持文档与代码同步更新

## 📝 贡献指南

如需添加新文档，请遵循以下步骤：

1. 确定文档类型，放入对应分类文件夹
2. 使用规范的文档格式
3. 在本 README 中添加文档索引
4. 更新文档统计信息
