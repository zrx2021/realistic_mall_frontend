## 📝 文档维护

### 添加新文档

1. 确定文档分类，选择对应文件夹
2. 创建符合命名规范的文档
3. 在 [README.md](README.md) 中添加索引
4. 在本文档中更新目录结构

### 文档分类规则

| 文档内容 | 放置位置 |
|---------|---------|
| 功能模块文档 | 创建对应模块文件夹 |
| 产品需求文档 | `产品需求/` |
| 技术方案文档 | `技术方案/` |
| 数据库设计文档 | `数据库设计/` |
| API接口文档 | 对应功能模块文件夹 |

### 命名规范

- **README-xxx.md** - 功能说明文档
- **PRD-xxx.md** - 产品需求文档
- **xxx-设计总结.md** - 设计总结文档
- **xxx-快速开始.md** - 快速开始指南
- **xxx-交付清单.md** - 交付清单文档
